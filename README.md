# 🗺️ Gezginin Rotası (Traveler's Log) — 175 kg 🏆

**Naim Agentic Iterative Mobile (NAIM)** Challenge kapsamında geliştirilen, "Cep Herkülü" **Naim Süleymanoğlu**'ndan ilham alan devasa özelliklere sahip seyahat asistanı. 🏅🌸✨

![Naim Süleymanoğlu](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c2/Naim_Suleymanoglu_1988.jpg/250px-Naim_Suleymanoglu_1988.jpg)

> *"Küçük telefon, dev güç."* — Bu uygulama, 5 kg'lık temel bir arayüzden başlayıp her iterasyonda yeni bir "ağırlık" (özellik) kaldırarak tam **175 kg (Super Heavyweight)** kapasiteye ulaştı!

---

## ✨ Öne Çıkan Özellikler (Highlights)

- **🤖 AI Seyahat Danışmanı:** Her şehir için özel seyahat ipuçları üreten akıllı asistan.
- **🇹🇷/🇬🇧 i18n Desteği:** Tek tuşla Türkçe ve İngilizce dilleri arasında geçiş.
- **📸 Görsel Hafıza:** Rotalarınıza galeriden fotoğraf ekleme ve anılarınızı görselleştirme.
- **🌙 Dinamik Karanlık Mod:** Göz yormayan şık karanlık tema (Dark / Light Mode).
- **🔍 Akıllı Arama:** Kayıtlı rotalar arasında şehir ismi veya notlara göre anlık filtreleme.
- **📜 Motivasyonel Alıntılar:** Harici API (Quotable) ile her rotaya ilham verici seyahat sözleri.

## 🚀 Teknolojik Araç Kutusu

- **Framework:** React Native + Expo SDK 54
- **State Management:** React Hooks (useState & useEffect)
- **Persistance:** AsyncStorage
- **Design:** Flowery, Sparkling & Relaxing Theme (Custom CSS)
- **APIs:** Quotable.io, Expo Image Picker, i18n Logic

## 🔁 NAIM Loop (Geliştirme Döngüsü)
